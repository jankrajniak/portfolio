## Description
- This project is a single-page React app of my personal portfolio
- The project additionally uses Bootstrap for styling (along with custom CSS)


## Installation Instructions
- The package.json and the vite.config.js files are included in the repository. 
- Run npm install to install all required dependencies
- Run "npm run dev" to run the app on localhost:3000 if making changes/editing
